

## **宿舍管理系统**


<br>

**一个基于 SpringBoot&Vue 的宿舍管理系统**

> 来自B站 [Spring Boot+Vue宿舍管理系统](https://www.bilibili.com/video/BV1YS4y1v7hL)


<br>
<br>


**部署运行：**

下载源码，拖入Idea打开.

分为前端后端两部分.

<br>

**SpringBoot后端：**

> 导入直接运行即可

<br>

**Vue前端：**

> `npm install`

> `npm run serve`

<br>


后端地址：[localhost:9090](http://localhost:9090/)

前端地址：[localhost:9999](http://localhost:9999/)

<br>
